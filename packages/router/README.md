## Installation

> In case of using GitHub package registry,
> the package name will be `@foxifyjs/router`.

### NPM

```shell
npm i @foxify/router
```

### PNPM

```shell
pnpm add @foxify/router
```

### Yarn

```shell
yarn add @foxify/router
```
