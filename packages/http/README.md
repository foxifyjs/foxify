## Installation

### NPM

```shell
npm i @foxify/http
```

### PNPM

```shell
pnpm add @foxify/http
```

### Yarn

```shell
yarn add @foxify/http
```

## Usage

```typescript
import { Request, Response, METHOD } from "@foxify/http";
```
